## TB6612&压降模块

| 引脚 | 连接对象 |
| --- | --- |
| **PWMA** | MCU 定时器 PWM 输出孔 |
| **AIN2** | MCU 普通 GPIO |
| **AIN1** | MCU 普通 GPIO |
| **STBY** | MCU 3.3V/5V 或 GPIO 置高 |
| **BIN1** | MCU 普通 GPIO |
| **BIN2** | MCU 普通 GPIO |
| **PWMB** | MCU 定时器 PWM 输出孔 |

### 2. 信号反馈排针 (Output_IO)

*用于将底层传感器信号回传给单片机。*

| 引脚 | 连接对象 |
| --- | --- |
| **E2B** | MCU 捕获/中断引脚 |
| **E2A** | MCU 捕获/中断引脚 |
| **E1B** | MCU 捕获/中断引脚 |
| **E1A** | MCU 捕获/中断引脚 |
| **ADC** | MCU 的 ADC 采样孔（不用可悬空） |
| **GND** | **必须接 MCU 的 GND** |
| **5V** | MCU 或传感器的 5V 供电孔（可选） |

### 3. 电源与电机物理接口

*用于接线大电流动力设备。*

| 接口名称 | 位置/形态 |
| --- | --- |
| **电源输入端子** | 顶部蓝色接线柱 |
| **电机 A 接口** | 右侧白色 6P 座 |
| **电机 B 接口** | 右侧白色 6P 座 |
| **电源输出端子** | 底部蓝色/排针 |
| **电源开关** | 顶部物理拨动开关 |

## 24H 自动行驶小车 (H题) - 硬件连接拓扑设计

根据题意要求（无需摄像头，沿 1.8cm 细线行驶且具备声光提示），并结合上述硬件清单，为你设计了严谨无冲突的引脚分配。考虑到清单中还包含陀螺仪 MPU6050，且串口屏自带蜂鸣器，本方案已最大化地利用了单片机的专属外设资源：

### 1. 🖥️ 淘金驰串口屏 (人机交互 + 声音提示)
利用 UART 串口通讯，并**复用屏上的蜂鸣器**进行“声光提示”中的声音部分，节省了 MCU 的额外引脚。
* **VCC** -> `5V` 供电孔
* **GND** -> `GND`
* **RX (屏接收)** -> `PA8` (UART_TX)
* **TX (屏发送)** -> `PB23 / PA9*` (UART_RX，需确认 LaunchPad 上 J14 跳线帽已使能)

### 2. 🛞 TB6612 电机驱动 (逻辑控制 Input_IO)
用于控制小车左右两个底盘差速电机。
* **PWMA (左轮调速)** -> `PB20` (PWM)
* **AIN1 (左轮方向 1)** -> `PB6` (GPIO)
* **AIN2 (左轮方向 2)** -> `PB7` (GPIO)
* **PWMB (右轮调速)** -> `PB13` (PWM)
* **BIN1 (右轮方向 1)** -> `PB16` (GPIO)
* **BIN2 (右轮方向 2)** -> `PB15` (GPIO)
* **STBY (驱动使能)** -> `3V3` (或直连逻辑高电平)

### 3. 👁️ 4路灰度传感器
用于 1.8cm 黑色圆弧寻迹。可以直接接 ADC 分组的引脚，既可配置为普通数字 GPIO（读取0或1），也可开启 ADC 采样进行模拟量平滑巡线。
* **S4 (最左探头)** -> `PA12` (ADC0.2)
* **S3 (中左探头)** -> `PA13` (ADC0.5)
* **S2 (中右探头)** -> `PA17` (ADC0.7)
* **S1 (最右探头)** -> `PA25` (ADC0.3)

### 4. 📐 GY-521 (MPU6050 姿态传感器)
由于圆弧行驶时单靠 4 路灰度循迹极易因为速度过快冲出黑线，加入 MPU6050 结合 PID 偏航角计算，可以极大提升圆弧弯道的行驶稳定性。
* **VCC** -> `3V3` (GY-521 模块内置稳压，推荐接 3.3V)
* **GND** -> `GND`
* **SCL** -> `PB2` (I2C_SCL)
* **SDA** -> `PB3` (I2C_SDA)
*(注：AD0默认接地地址为0x68，INT/XDA/XCL等引脚悬空即可)*

### 5. 📏 闭环电机编码器反馈 (Output_IO)
电机带编码器测速，是电赛控制题（尤其是定距离行驶和精准停车）的制胜法宝。
* **E1A (左轮 A 相)** -> `PB19` (利用 ADC1.6 端口作为带中断功能的 GPIO)
* **E1B (左轮 B 相)** -> `PB18` (利用 ADC1.5 端口作为普通输入)
* **E2A (右轮 A 相)** -> `PB7` (SPI_POCI，复用为带中断功能的 GPIO)
* **E2B (右轮 B 相)** -> `PB8` (SPI_PICO，复用为普通输入)

### 6. 🚨 状态反馈 (LED 提示)
用于任务中“光”提示的要求。
* **LED 控制 (蓝光)** -> `PB22` (利用 LaunchPad 板载 RGB LED2 的蓝色通道。已查阅官方手册，需确保 **J5** 跳线帽已安装。这样无需再外接发光二极管，直接用单片机控制 PB22 即可实现定点闪灯提示)

> **💡 拓扑避坑指南**：
> - **电机电源独立**：TB6612 的电机电源输入（顶部蓝色接线柱）必须直接连电池，**绝对不能**由单片机取电，否则会直接烧毁单片机！
> - **共地要求**：传感器的 GND、TB6612 的 GND，以及单片机的 GND，全系统必须共地。
